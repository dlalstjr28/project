## final_project.zip
도커 프로젝트를 통한 3-tier 어플리케이션 운영(영화 정보 제공사이트 만들기 + 자동실행하는 크롤링 컨테이너)
사용한 웹 프레임워크는 django (web front, web backend)
사용한 저장소는 mysql
다운 받은 다음에 폴더에서 docker-compose up --build 를 하게 되면 실행되는 구조로 만듬.
